# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-02-23

### Resume Page — UI/UX Improvements

#### Skills Section
- Improved bullet point spacing for better readability
- Replaced default browser list markers with custom styled bullets for visual consistency across all sections

#### Projects Section
- Expanded layout to full page width, matching the experience section
- Increased bullet point spacing for improved readability
- Styled each project as a distinct card with its own background, border, and hover effect for easier visual distinction
- Added responsive two-column grid layout for project cards on larger screens

#### Responsive Color Theming
- Standardized heading (`h3`) and bullet point colors across all resume sections
- All `h3` headings and bullet markers now consistently transition from `#ff6600` (mobile) to `#b80` (desktop ≥ 1024px)
- Resolved specificity issues that prevented certain headings from updating at the desktop breakpoint

## [1.0.0] - Initial Release

- Launched personal portfolio website with home, resume, projects, and contact pages
- Mobile-first responsive design using vanilla HTML, CSS, and JavaScript
- Hosted on GitHub Pages
